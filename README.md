# my-app

Practical assessment for the Frontend Developer Internship at CodesRaft.

Built with React 19, TypeScript, and Vite. UI components from shadcn/ui, styled with Tailwind CSS. Animations handled by GSAP and smooth scrolling with Lenis. Routing via React Router.

Setup instructions:

# clone the repo
git clone <repo-url>
cd app

# install dependencies
npm install

# start dev server
npm run dev

# build for production
npm run build

# preview production build
npm run preview
